SMARTKEY USB DRIVER FOR WINDOWS 9x/Me/2000/XP
---------------------------------------------

Driver for the USB SmartKey in Windows 9x/Me/2000/XP


INSTALLATION
------------

To install the driver please follow these steps:

- Decompress this archive in a temporary directory
- Plug the SmartKey USB in a USB port
- When Windows asks for the driver select the directory containg the SKEYUSB.INF file
  and automatically Windows will install the driver
